## opus 終審實測驗過(逐點)
- **判定式單源**:pre-push/Stop/verdict 三處都經 `code-loop check`,無某處自算 tier 漂移(pre-push 顯示行重算但同一 deterministic 命令,不影響 block 決策)。
- **綁 HEAD**:write/compare 皆 full sha(`git rev-parse HEAD`),full-vs-full 無 off-by;branch 切換走 per-branch 檔名;情境 4 證 HEAD-moved → stale → blocked。
- **hook 閉環**:`t_hook_copy_list_completeness`(註冊⊆複製 set-diff)於 e9e61ae 前會紅、後綠——真通則守衛防「註冊了沒複製」silent no-op 復發(C1/T12 同型)。
- **fail-open**:rc=2 / 缺 lumos / subprocess 爆 皆不擋不 crash(測試 E + hook 8/9)。

## 誠實天花板
- **非 oracle**:Stop 注入可被無視、pre-push 可 `--no-verify`。關得掉「忘了看」(Stop 推到眼前)、「隨手漏」(pre-push 硬擋);關不掉「刻意繞+不誠實」——lumos 不能替你 spawn agent 跑 code-loop、git 不防 --no-verify。同 [[主動影響幅度偵測_計劃]]/design-loop「push+摩擦+地板,非 oracle」。
- **tier=high 假陽** → `code-loop skip --note` 是逃生閥(不卡死、留痕可審)。
