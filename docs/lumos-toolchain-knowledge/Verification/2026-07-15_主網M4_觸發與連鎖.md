# 驗證：關係層主網 M4——觸發與連鎖（整包收官）

主網終站（[[關係層主網_實作計畫]] M4/[S1][S2][S3][S4]）——把 M1 決策 ID、M2 typed 索引、M3 帳本接成**活的主網**：翻案的當下，受波及的鄰居當場被攤到眼前。

## 變更範圍（scripts/lumos）
- **`_typed_hop1(env, node, idx=None)`**：typed hop-1 共用原語（M2 索引上單層列舉、只走 verified_by/plan_refs、related 恆不展）；resume 重構改吃它（DRY）＋菱形不重展 stderr 留痕。
- **`cascade_surface(env, node, root_gid, stream)`**：surfacing 印行器（NEIGHBOR/NEIGHBOR-AMBIG 行；dispatcher 印 CASCADE 行——T2 職責切分）。
- **[S1] dispatcher 觸發**：supersede rc=0 → 獨立 try 包「②`rel_cascade_create` 建檔＋③surfacing」→ stderr 印 CASCADE+NEIGHBOR；決策無 id → `CASCADE-SKIP`（不建檔）；失敗 → `CASCADE-ERROR` **fail-open**（rc 仍 0、不回滾、補網 E2 兜底）；無鄰居 → `CASCADE-EMPTY`。
- **`impact --node <節點>`（CLI 皮＋`--json`）**：第三模式，與 `--file`/`--diff`/`--depth` 互斥 rc=2；既有模式與 PreToolUse hook 熱路徑零觸碰（[[主動影響幅度偵測_計劃]] 的工具面新增能力、不改既有語意）。

## 天花板（整包收官重申）
- **判斷閘是 GIGO**：工具保證攤鄰居＋留痕誰判的；判得對不對靠 AI/人（不對稱紀律：confirm 放寬/prune 保守留痕/拿不準升人）。
