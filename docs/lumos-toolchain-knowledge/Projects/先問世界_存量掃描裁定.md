# 「先問世界」存量掃描裁定(2026-07-07)

三路平行深度研究(WebSearch/WebFetch 真讀官方 docs、GitHub 原始碼、Swimm patent US11132193B1、學術文獻 Offutt 1996/Kurtz FSE 2016),對照 `scripts/lumos` 實碼逐項核實 lumos 現況後裁定。緣起:[[自主loop加法偏食]] 的外延——SDD 管線沒有「先問世界」步驟(已同步制度化為 PRIOR-ART 三問)。**家規:borrow-design(借精神原生實作,零依賴不變),不 adopt。**

### 中(1 條,唯一需正式 design-loop)
**suspect 語意(兩層合一)**——「世界變了 code 沒變」的機械解,lumos 唯一完全空缺的漂移語意:
- **邊級(Doorstop)**:Verification 寫入/重驗時記「被驗 Systems 的 summary block sha8」進 `governance/link-stamps.json`;doctor 比對現行 hash → 不符=suspect 驗證;`lumos stamp clear` 明確重蓋章。血換教訓:**只 hash 載重欄位**(否則 typo 修正全圖 suspect)。
- **行級(Swimm patent)**:節點引用 `path:line` 時存行內容 hash+摘錄 baseline;doctor 重驗:內容不變=綠;**變了但 difflib 能重定位同 snippet=自動修行號(吸收最大宗假陽性「只是搬家」)**;定位不到=suspect;+ `dismiss --note` 台帳(Swimm 自己沒有的逃生口,避免強制 review 疲勞)。
- Swimm 證明**不用 AST、純 diff+內容比對堪用**——正對零依賴體質。動 anchor/Check P 合約邊界 → 過 design-loop。

## Borrow-later(7 條)
self_audit 日期→summary-hash(Doorstop review stamp;suspect ledger 建好順手)/ 未覆蓋 Systems 檢查+`coverage: exempt` 豁免旗標(豁免先行否則全噪音)/ `.lumos/warnings.json` 專案自訂宣告式檢查(限三種斷言禁 eval;等第一個專案特有需求)/ findings 必附機器可驗座標(G1 擴到 review findings)/ golden replay 校準 auditor(等 golden 語料 10+ 份;外部人標錨,非已 rejected 的自證悖論)/ vendor-lock.json 記 SHA / hook-config 統一 CODE_EXTS 宣告。

## Skip(14 條,理由同等重要)
需求層級樹、traceability matrix 視圖、查詢語言、ROLE 標籤、環偵測、Proposed/Rejected 狀態(計劃節點+alternatives_considered 已更好)、code 檔 sha 引用(天天 suspect 必噪)、staged-stash、fail_fast、parallel gates、coverage-guided 選測試(3-5 抽樣不需要)、**LLM 判 review effort(risk-tiered 機械分級已覆蓋,LLM 判是倒退)**、**constitution SemVer 修憲(Check D 機械守衛對單人+AI 強於治理儀式)**、**delta-spec 格式(第二真相源,犯圖譜唯一真相禁忌)**、inter-rater κ(n 太小)、Swimm auto-rewrite 散文(它自己都只敢改機械錨)。

## 誠實反向發現(lumos 領先處)
- **決策時效性**(`valid_under`/`revalidate_when`/`stale --match`/ADR 四欄位):領先全部研究對象——整個 ADR 生態沒人做「accepted 之後還算不算數」。
- Check T 認真方法(discover 白名單)強於 StrictDoc 的 marker 存在性檢查。
- 反向 impact(code→節點)是 StrictDoc 沒有的。

## 貫穿發現(體質印證)
成熟工具三共識,lumos 全數同宗:①**狀態存工件本身**非外部 DB(PR-Agent comment 錨/Swimm .swm 進 repo ↔ lumos governance/ 台帳)②**機械先驗、LLM 後判**(Swimm 不用 AST/CodeRabbit sandbox 先 grep ↔ lumos refcheck/pitfalls)③**抑噪走 context 注入**非硬過濾表(CodeRabbit learnings ↔ FP 台帳移植時守住此語意)。
